RESPONSIVE BEHAVIOR
================================================================================

DESKTOP (> 768px)
  ├─ Sidebar always visible (280px width)
  ├─ Menu items expanded based on localStorage state
  ├─ Hover states on menu items
  └─ Full text labels visible

TABLET (640px - 768px)
  ├─ Sidebar visible but icons only
  ├─ Hover to expand full labels
  ├─ Collapsible sections on tap
  └─ Touch-friendly tap targets (44px min)

MOBILE (< 640px)
  ├─ Sidebar collapses by default (Menu icon in top bar)
  ├─ Tap Menu icon to toggle sidebar
  ├─ Sidebar overlays main content
  ├─ Tap a menu item to navigate and close sidebar
  └─ Full text labels visible when sidebar open


STYLING REFERENCE
================================================================================

COLORS (CSS Variables)
  --color-text-primary           Main text color
  --color-text-secondary         Secondary text (labels)
  --color-text-tertiary          Tertiary text (descriptions)
  --color-surface-default        Menu background
  --color-surface-light          Hover state
  --color-bg-primary             Content background
  --color-bg-secondary           Active state
  --color-accent                 Highlight color
  --color-warning                Warning indicator
  --color-warning-soft           Warning background
  --color-border-light           Border color

TYPOGRAPHY
  Section Headers: 12px uppercase bold
  Menu Items: 14px regular
  Badges: 10px medium/bold
  Descriptions: 12px regular (optional)

SPACING
  Section padding: 12px vertical
  Item padding: 12px (all sides)
  Icon size: 18px
  Badge padding: 1.5px 1.5px (0.375rem)
  Gap between icon/label: 12px


ICON SIZING
================================================================================

Navigation Icons: 18px
Section Headers: 16px (chevron)
Top Bar Icons: 20px
Badge Icons: 16px


STATE INDICATORS
================================================================================

Active Route
  ├─ Background: var(--color-bg-secondary)
  ├─ Text Color: var(--color-text-primary)
  └─ Visual Cue: Highlighted background

Expandable Section (Collapsed)
  ├─ Chevron: Points left (-90°)
  └─ Opacity: 60%

Expandable Section (Expanded)
  ├─ Chevron: Points down (0°)
  └─ Opacity: 100%

Badge Types
  ├─ AI: Yellow/warning background
  ├─ New: Red/error background
  ├─ Soon: Purple background
  └─ Count: Gray background


INTERACTION FLOWS
================================================================================

CLICKING MENU ITEM (Desktop)
  1. Item highlighted with background change
  2. Navigation occurs immediately
  3. Active state persists on new page
  4. Scrolls menu to show new item if needed

CLICKING MENU ITEM (Mobile)
  1. Item highlighted
  2. Navigation occurs
  3. Sidebar closes automatically
  4. Back button available in header

TOGGLING SECTION (Desktop)
  1. Chevron animates 90° rotation
  2. Section content animates in/out
  3. State saved to localStorage
  4. Animation duration: 200ms

RESPONSIVE COLLAPSE (Window resize)
  1. Sidebar detection on mount
  2. Automatic collapse on < 768px
  3. Manual toggle always available
  4. State persists across page navigations
