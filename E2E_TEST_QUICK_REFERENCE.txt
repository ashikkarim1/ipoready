================================================================================
                            SEVERITY LEVEL GUIDE
================================================================================

P0 (CRITICAL - BLOCKING)
  └─ Prevents core feature from working
     Examples: Cost Calculator won't save, Dashboard data incorrect, Export fails
     Action: Must be fixed before launch

P1 (MAJOR - HIGH PRIORITY)
  └─ Significant impact but has workaround
     Examples: Consent letter formatting issue, Minor validation missing
     Action: Should be fixed before launch

P2 (MINOR - MEDIUM PRIORITY)
  └─ Low impact, cosmetic or edge case
     Examples: Typo in label, Rare edge case crash
     Action: Can defer to post-launch if necessary

P3 (TRIVIAL - LOW PRIORITY)
  └─ No functional impact
     Examples: Spacing, color consistency, hover text
     Action: Can defer to v1.1 or later
