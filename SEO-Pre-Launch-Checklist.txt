TROUBLESHOOTING GUIDE
================================================================================

Problem: GSC Domain Verification Fails
Solution:
  1. Verify DNS TXT record is exactly correct (copy/paste entire string)
  2. Wait 24-48 hours for DNS propagation
  3. Check DNS provider shows record as active
  4. Try alternative: HTML file verification method
  5. If still failing: Contact Google Search Console support

Problem: Sitemap.xml Not Being Submitted
Solution:
  1. Verify sitemap is accessible: https://www.ipoready.ai/sitemap.xml
  2. Check sitemap has valid XML structure
  3. Verify all URLs in sitemap are valid and accessible
  4. Check robots.txt isn't blocking /sitemap.xml
  5. Resubmit in GSC > Sitemaps section

Problem: Pages Not Appearing in Search Results
Solution:
  1. Check GSC Coverage tab - are pages indexed?
  2. If not indexed:
     - Verify pages are accessible (no 404)
     - Check robots.txt isn't blocking them
     - Request indexing via GSC > URL Inspection > "Request indexing"
  3. If indexed but not ranking:
     - Check competitor rankings for same keywords
     - Optimize page title and meta description
     - Improve Core Web Vitals if they're below target
     - Add internal links from other pages

Problem: Core Web Vitals Not Meeting Targets
Solution:
  1. Use PageSpeed Insights to identify issues
  2. Common issues:
     - LCP: Optimize images, lazy load, reduce JS
     - FID: Reduce JavaScript complexity
     - CLS: Add size attributes to media, avoid dynamic content shifts
  3. Vercel recommendations:
     - Enable Edge Caching
     - Use next/image for responsive images
     - Enable Static Export if applicable
  4. Test after each optimization in PageSpeed

Problem: Google Business Profile Verification Taking Too Long
Solution:
  1. Postcard verification: Normal timeline is 1-2 weeks
  2. Faster option: Request phone verification instead
     - Go to profile > Verify location > Call Google
     - Receive verification code by phone
     - Typically completes in 24 hours
  3. Ensure address is exact and verifiable
  4. Check business address is not considered "non-verifiable"
