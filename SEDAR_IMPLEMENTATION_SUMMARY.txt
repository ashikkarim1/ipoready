SEDAR-SPECIFIC REJECTION CODES HANDLED
=======================================

1. MISSING_SIGNATURE - Required signatures missing or invalid
2. INVALID_IFRS_FORMAT - Financial statements not IFRS compliant
3. CONSENT_NOT_SIGNED - Officer/director consents not properly signed
4. DOCUMENT_FORMAT_ERROR - Document doesn't meet SEDAR requirements
5. LANGUAGE_MISMATCH - Bilingual documents not properly formatted
6. PROSPECTUS_INCOMPLETE - Missing required prospectus sections
7. FINANCIAL_DATA_MISSING - Required financial data incomplete
8. EXECUTIVE_COMPENSATION_MISSING - Insufficient compensation disclosure
9. MD_A_MISSING - Management Discussion and Analysis required
10. AUDITOR_REPORT_INVALID - Auditor report doesn't meet standards

ERROR HANDLING STRATEGY
=======================

1. Validation Errors (Pre-submission)
   - Caught and returned as validation errors
   - Prevents submission if invalid
   - Clear field-specific error messages

2. API Errors (During submission)
   - Retry logic with exponential backoff
   - Maximum 3 retry attempts
   - Timeout: 30 seconds per request

3. Rejection Errors (Post-submission)
   - Mapped to SEDAR rejection codes
   - Actionable suggestions for fixing
   - Can be used for resubmission workflow


USAGE EXAMPLE
=============

import { SEDARAdapter } from '@/lib/filing-adapters'

// Initialize adapter
const sedarAdapter = new SEDARAdapter(
  apiKey: process.env.SEDAR_API_KEY,
  sandboxMode: true,
  language: 'en'
)

// Prepare filing
const filing = {
  id: 'FILING-001',
  status: 'draft',
  documents: [...],
  errors: [],
  metadata: {
    companyName: 'TechCorp Inc.',
    exchangeSymbol: 'TECH',
    // ... other metadata
  }
}

// Validate
const validation = await sedarAdapter.validate(filing)
if (!validation.isValid) {
  console.error('Validation errors:', validation.errors)
  return
}

// Submit
const result = await sedarAdapter.submitFiling(filing)
console.log('Filed successfully:', result.externalId)

// Track status
const status = await sedarAdapter.trackFilingStatus(result.externalId)
console.log('Status:', status.status)
