FEATURE DETAILS
================================================================================

INPUT PARAMETERS:
  - Company Revenue (USD)
  - Stock Exchange (NYSE, NASDAQ, TSX, ASX, OTHER)
  - Complexity Level (low, medium, high)
  - Timeline (months)

COST BREAKDOWN:
  1. Legal Costs
     - Exchange-based fees (1.0%-1.5% of IPO size)
     - Complexity multiplier (0.8x-1.3x)
  
  2. Accounting Costs
     - Base $80k + 0.05% of revenue
     - Complexity multiplier
  
  3. Underwriting Costs
     - 2% of estimated IPO size (15% of revenue)
  
  4. Printing Costs
     - Base $25k
     - +10% if timeline > 12 months
  
  5. Filing Costs
     - NYSE: $250k, Others: $180k
  
  6. Contingency
     - 10% of subtotal

OUTPUT VISUALIZATIONS:
  - Total cost display with percentage of IPO
  - Bar chart: Cost breakdown by category
  - Pie chart: Cost distribution percentages
  - Detailed table: Itemized costs
  - Benchmark comparison: Industry averages and ranges
  - PDF export button

USER INTERFACE:
  - Tailwind CSS v4 styling
  - Dark mode support (automatic)
  - Responsive design (mobile/tablet/desktop)
  - Lucide React icons
  - Form validation feedback
  - Loading states
  - Error handling
  - Proper spacing and typography

DEPLOYMENT STEPS
================================================================================

1. Apply database migration:
   neonctl sql -c <connection> < migrations/020_cost_calculator.sql

2. Verify dependencies installed:
   npm list recharts lucide-react react-hook-form zod

3. Start development server:
   npm run dev

4. Access calculator:
   http://localhost:3000/dashboard/financial-mgmt/cost-calculator

5. Test calculation:
   - Enter test data
   - Verify charts render
   - Test export functionality
   - Verify dark mode

6. Build for production:
   npm run build && npm start
